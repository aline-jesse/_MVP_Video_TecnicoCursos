🚀 Passo a Passo

1. Acesse a Aplicação

Clique no link abaixo para abrir o Estúdio IA de Vídeos:

🌐 Abrir Estúdio IA de Vídeos

2. Fazer Login como Admin

Na página inicial, clique em "Entrar" e use estas credenciais:

Email: admin@estudio.ai

Senha: 123456

💡 Dica: Qualquer senha com 3+ caracteres funcionará

3. Acessar o Painel Admin

Após o login, você verá o link do admin de duas formas:

  • Menu do Usuário: Clique no avatar (canto superior direito) → "Painel Admin"
  • URL Direta: http://localhost:3000/admin

🎛️ Funcionalidades do Painel Admin

🎨 Identidade Visual

  • • Upload de logo
  • • Upload de favicon
  • • Cores personalizadas
  • • Configuração de fontes

🏢 Informações

  • • Nome da empresa
  • • Subtítulo/slogan
  • • URL do site
  • • Email de suporte

⚙️ Avançado

  • • Política de privacidade
  • • Termos de serviço
  • • Exportar/Importar
  • • Preview em tempo real

🐛 Resolução de Problemas

Se não aparecer "Painel Admin" no menu:

  • • Confirme que usou o email: admin@estudio.ai
  • • Procure pelo ícone de escudo (🛡️) no dropdown do usuário
  • • Tente fazer logout e login novamente

Se receber erro 403 (Acesso Negado):

  • • Verifique se está usando o email admin correto
  • • Limpe os cookies do navegador
  • • Reinicie a sessão
